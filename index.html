算力收租权的开始

从 build.nvidia.com 免费送 token 这件"小事"，拆解 NVIDIA 正在发生的五层蛋糕战略——以及它意味着的一次估值重构。

2026.04.23 · 投研长文
▌ TL;DR

NVIDIA 不是在送 token，而是在 用硬件利润补贴"软件+服务"生态——这是 NVIDIA CEO 黄仁勋亲口讲过的"AI 工厂五层蛋糕"在 2026 年的真实落地。免费 token 踩在蛋糕最上层（Services），真正目的是把算力从"一次性销售"改造成"持续计费的订阅管道"。这件事对 NVIDIA 的意义不是多赚几十亿 token 钱，而是 把自己从周期性半导体公司重估为 AI 平台公司——参照苹果 2015-2024 年服务化路径，这可能带来 $5,000 亿级别的估值溢价。副作用是：CoreWeave 这类"纯 GPU 二道贩子"的生意模式面临结构性压力。

Ⅰ.为什么免费？——不是慈善，是获客

一个直接问题：NVIDIA 每月给开发者免费 1000 次推理请求，这到底值多少钱、为什么要送？

按 build.nvidia.com 当前开放的几款模型（DeepSeek V3、Kimi K2、Llama 3.3、MiniMax M2 等）推算，假设平台 100 万活跃开发者、每人每月用满 1000 次、单次平均 2K tokens，一年消耗约 24 万亿 tokens。按 NVIDIA 自用推理成本约 $0.3/M tokens 估算，直接算力成本约 $7 亿/年。加上约 2 万张 GPU 折旧、工程团队、DevRel 驻场支持、市场费用，一年真金白银投入约 $10-15 亿美元。

这笔钱对 NVIDIA 是什么概念？FY2025 年净利润 $730 亿美元，这笔投入占净利润 1.5%——基本是会计误差量级。

那么 NVIDIA 花这 1.5% 换什么？

答案有四层，按重要性从低到高：

第一层：短期 token 付费转化（小钱）。免费额度耗尽后的企业用户直接按量付费。这部分未来 3 年可能到 $20-30 亿/年，对 NVIDIA 规模不算大。

第二层：硬件销售导流（中钱）。开发者在 build.nvidia.com 上用 NVIDIA 推理习惯后，他们所在的公司做 POC、上生产时必然首选 NVIDIA GPU。这是传统意义的"获客漏斗"——但拉动的是高毛利硬件销售，间接收益远超 token 费。

第三层：生态锁定（大钱）。CUDA 已经锁定了训练生态，build.nvidia.com + NIM 要锁定推理生态。一旦百万开发者习惯用 NVIDIA 的 API 端点、用 TensorRT-LLM 优化过的模型，切换到 AMD/Google TPU 的迁移成本会变得极高。生态锁定本身不直接产生收入，但它决定了 NVIDIA 未来 10 年的定价权。

第四层：估值模型重构（最大的钱）。这是最被低估但最重要的一层——免费 token 只是一个动作，背后是 NVIDIA 从"卖 GPU 的半导体公司"重新定位为"AI 平台公司"的叙事工程。资本市场给这两种公司的估值倍数完全不同，差距可能是 $5,000 亿美元级别。这一点在第 Ⅳ 章和第 Ⅴ 章详细展开。

免费 token 的真实性质：NVIDIA 用 1.5% 的净利润，同时购买"短期 token 收入 + 硬件销售漏斗 + 生态锁定 + 估值重构叙事"四重回报。这不是慈善，是 黄仁勋算过账的投资。

Ⅱ.什么模式？——黄仁勋五层蛋糕的现实落地

黄仁勋在过去几次 GTC 和财报电话会上反复讲过一个概念："AI Factory is a full-stack layered business"——AI 工厂是一个自下而上、五层堆叠的生意。这五层分别是：

逐层映射——免费 token 是整盘棋的哪一步

第 1 层 Chips（芯片）：NVIDIA 的传统地基，已经赚了十几年，FY2025 数据中心收入 $1,150 亿主要来自这里。护城河来自 Hopper/Blackwell/Rubin 的制程与架构领先。但这层是一次性销售 + 周期性——客户买完就结束，下一代新品出来要重新销售。

第 2 层 Systems（系统）：不卖单卡，卖 DGX / HGX / MGX 整机——GPU + CPU + 内存 + 散热的一整台。毛利比单卡略低，但绑定度更高。这层本质还是硬件一次性销售，但卖的是"交钥匙方案"。

第 3 层 Networking（网络）：收购 Mellanox 之后的成果。NVLink、InfiniBand、Spectrum-X——把单机 GPU 扩展到万卡集群的粘合剂。这层对大客户（如 xAI Colossus、Meta 30 万卡集群）至关重要，竞争者基本追不上。仍属硬件销售。

第 4 层 Software（软件）：从 CUDA 起步，延伸到 AI Enterprise Suite（$4,500/GPU/年订阅）、TensorRT-LLM（推理加速）、NIM（模型容器）。这是 NVIDIA 过去 18 年最深护城河——软件积累是其他硬件厂追不上的部分。这层收入开始体现订阅属性，但规模仍然小（FY2025 官方披露软件 run-rate 约 $20 亿）。

第 5 层 Services（服务）：DGX Cloud（租用 NVIDIA 自己建的数据中心）、build.nvidia.com（API 平台）、NIM 托管服务。这是最新、最小、但增长最快、毛利最高的一层。免费 token 就是这一层的"获客广告"。

这个结构对标谁？

五层蛋糕的设计几乎就是"英特尔 + 微软 + AWS"三合一——Intel 做硬件、Microsoft 做软件、AWS 做服务，历史上是三家公司各自称霸一层，彼此互为客户与竞争。黄仁勋的野心是一个人吃完三层。

但这个野心只有 NVIDIA 能做成，因为：

  • 硬件层 + 软件层：CUDA + GPU 的绑定度比 Intel + Windows 更深——迁移 CUDA 意味着重写代码+重训模型，企业级迁移成本 $1000 万起。
  • 软件层 + 服务层：NIM 作为推理容器格式，既是软件产品又是服务入口。客户用 NIM 部署等于自动消耗 NVIDIA 软件授权 + 云服务。
  • 三层协同：硬件卖得越多 → 开发者越多 → 软件生态越厚 → 云服务需求越大 → 更多硬件销售。这是一个自我强化的飞轮。
免费 token 的战略意义：NVIDIA 正在从"第 1 层主导"向"第 5 层延伸"，目标是让收入结构从 "硬件 88% + 其他 12%" 演化到 "硬件 65% + 软件服务 35%"。这不是简单的品类扩张，是从硬件公司 → 平台公司的身份切换。

Ⅲ.对谁好、对谁坏？——生态四方博弈

免费 token 不是孤立事件，它改变了 AI 基础设施的格局。拆开看四方影响：

✅ 最受益方：开发者与中小企业

这是 NVIDIA 明牌讨好的对象——他们拿到真实利益：

  • 免注册成本、免部署成本、免运维成本，直接调用顶级开源模型
  • 一个 API Key 同时访问 DeepSeek / Kimi / Llama / Mistral，比自己搭 vLLM 集群省事百倍
  • 做 POC、原型、小流量生产，基本"够用"

但这是短期甜头。开发者一旦习惯 NVIDIA 的 API 风格、认 NVIDIA 的模型优化版本，未来做生产部署时会自然选 DGX Cloud 或本地 NVIDIA 集群。今天的免费是明天的锁定。

⚠️ 中度受益但隐含风险：开源模型方

代表：DeepSeek、Moonshot（Kimi）、MiniMax、Mistral、Meta Llama。

他们接入 NVIDIA 平台的动机很直接：

  • 零成本分发：不用自己建 API 基础设施就能触达百万开发者
  • NVIDIA 背书：能上 NVIDIA 货架等于行业认证，对中国模型出海尤其重要（DeepSeek 去年的"全球现象级"很大程度来自 NVIDIA 首页推荐）
  • 免费推理优化：NVIDIA 工程师帮做 TensorRT-LLM 量化和 Kernel 优化，成果模型方可拿回自用

隐含的风险是：用户关系被 NVIDIA 拿走。开发者用 DeepSeek V3，但调用的是 NVIDIA API，DeepSeek 不知道谁在用、用来做什么。长期看，模型方被降维为"给 NVIDIA 平台供货的内容生产商"——就像 App 开发者和苹果的关系。

❌ 明显受损方：AWS / Azure / GCP 的推理业务

三大云厂之前的算盘是：自己向 NVIDIA 采购 GPU → 自己搭推理服务（Bedrock / Azure AI Foundry / Vertex AI）→ 直接对开发者收费。NVIDIA 是零部件供应商，不能直接接触终端开发者。

免费 token 破坏了这个分工——NVIDIA 直接绕过云厂接触开发者，把"开发者 → 云厂 → NVIDIA"的三级分工压缩成"开发者 → NVIDIA"两级。云厂的中间商利润被侵蚀。

所以你会看到云厂的反制：

  • AWS：自研 Trainium / Inferentia 芯片，降低对 NVIDIA 依赖
  • Azure：深度绑定 OpenAI 独家，把闭源顶级模型圈在自己平台
  • Google：TPU + Gemini 全栈自研，连硬件都不用 NVIDIA

这是一场长期拉锯战。云厂短期不敢翻脸（还要买 Blackwell/Rubin），但会持续加码去 NVIDIA 化。

❌❌ 最受伤方：纯 GPU 租赁商（CoreWeave、Lambda、RunPod）

这是最容易被忽略但影响最直接的一方——详见第 Ⅵ 章单独展开。简单说：他们的生意模式被 DGX Cloud + build.nvidia.com 直接替代。

NVIDIA 自己：最大受益方，但也承担最大风险

受益：生态锁定 + 估值重构（见第 Ⅳ、Ⅴ 章）。

风险：

  • 反垄断：美国 DOJ、欧盟已多次约谈 NVIDIA，重点是 CUDA 绑定和 GPU 配额分配。把硬件+软件+服务全包可能触发拆分诉求。
  • 客户反水：如果云厂觉得 NVIDIA 抢食太狠，集体加速 TPU / Trainium / Maia 投入，NVIDIA 硬件销售可能受压。Microsoft 2025 年已减少部分 NVIDIA 采购转向 Maia 自研——这是危险信号。

Ⅴ.对照苹果——服务化的意义是什么

第 Ⅳ 章说"估值溢价是大头"——这个说法需要证据。苹果 2015-2024 年的服务化路径就是最好的参照，它证明了"硬件公司成功做服务"的估值魔法是真实存在的。

苹果发生了什么——事实回顾

时点硬件收入服务收入服务占比PE (Forward)市场定性
2015$2,105 亿$199 亿9%~12x硬件周期股
2019$2,140 亿$463 亿18%~22x过渡期
2024$2,940 亿$960 亿25%~32x消费科技平台

核心事实：2015-2024 年，苹果的 PE 倍数从 12x 涨到 32x，涨了 2.7 倍。同期硬件业务基本没涨（从 $2,105 亿到 $2,940 亿，10 年仅 40%），所有的估值跃迁都来自服务业务的从无到有。

为什么服务化会带来估值重估？——四个结构性改变

苹果的例子说明，市场重估硬件公司的依据有四条，缺一不可：

改变 1：收入可预期性。硬件销售是"这季度订了多少单"，服务订阅是"这批用户每月自动续费"。后者的报表噪音低一个数量级，机构投资者敢重仓。

改变 2：毛利率抬升。苹果产品线毛利 36%，服务毛利 74%——混合毛利随服务占比上升年年涨。苹果 2015 年整体毛利 40%，2024 年 46%。

改变 3：护城河加深。买 iPhone 的人不一定买下一台 iPhone，但订了 iCloud+Apple Music 的人几乎不会取消。订阅 = 零流失率 = 永续现金流。

改变 4：增长再生性。硬件是一次性的，卖完就结束；服务是复利的，用户基数越大、单用户支出越高、老用户续费时间越长。苹果服务收入 2015-2024 年 CAGR 19%，同期硬件 CAGR 只有 3.5%。

NVIDIA 的对照——四条都具备

苹果四项结构性改变NVIDIA 对应实现概率
收入可预期（订阅）DGX Cloud + NIM 订阅 + AI Enterprise✅ 高
毛利率抬升（服务 > 产品）软件毛利 85%+ > 硬件 75%✅ 极高
护城河加深（生态锁定）CUDA + NIM 双重锁定，比 iOS 更深✅ 极高
增长再生性（复利）Token 消耗跟随企业 AI 渗透率年年增长✅ 高

NVIDIA 在每一项上都不比苹果差，有些甚至更强。

NVIDIA 独有的三个加成——可能比苹果更狠

加成 1：B 端天花板 >> C 端天花板。苹果服务收入的上限是全球消费者的娱乐/生产力预算（可乐式市场）；NVIDIA 服务收入的上限是全球企业的 IT + AI 预算——按 Gartner 数据，2030 年全球企业 AI 支出预计 $1.5 万亿，比苹果服务的理论上限大 10 倍以上。

加成 2：选择权 vs 必选项。消费者可以不订 Apple Music（听网易云也行）；但企业无法不用 CUDA——训练好的模型要么跑在 NVIDIA 上，要么重写代码花千万美金迁移。这不是加成一点，是加成一个数量级。

加成 3：生态飞轮更强。苹果生态：开发者写 App → 用户买 iPhone。NVIDIA 生态：研究员用 CUDA 发论文 → 企业按论文找人才 → 人才只会 CUDA → 企业买 NVIDIA。从学术到产业的全链条锁定，苹果在开发者教育上做不到这么深。

关键差异——NVIDIA 的"估值起点"更高

但事情没那么美好。苹果 2015 年开始做服务时，市场给它的 PE 是 12x——起点便宜。NVIDIA 现在还没真正兑现服务收入，市场给的 PE 已经 35x——服务化的预期已经提前 price-in 了一半。

这意味着 NVIDIA 的投资逻辑比苹果当年更精细：

  • 如果服务 run-rate 按预期兑现（年增 30%+，3 年内软件占比破 15%）：PE 维持 35x 不回落，股价继续涨（对比不做服务化会跌回 20x 的场景）。
  • 如果服务 run-rate 不及预期：市场会惩罚"过度 price-in"，PE 回落到 25x 甚至 20x。
苹果对照的真实意义：不是 NVIDIA 一定能复制苹果的 PE 2.7x 涨幅，而是"硬件公司做服务"的估值魔法有真实历史案例支撑。对 NVIDIA 投资者而言，真正的 alpha 不在于"会不会成功做服务"，而在于 服务 run-rate 兑现的速度——慢了，当前估值透支；快了，估值可能再上一层。

Ⅵ.副作用：CoreWeave 会被打成什么样

这一章讲最被低估的一个事：NVIDIA 的收租化会直接冲击 CoreWeave 这类"纯 GPU 租赁"公司的商业模式。

CoreWeave 的生意是什么

CoreWeave 的业务模式可以一句话概括："从 NVIDIA 批量买卡，改造成云服务再转租给 AI 客户"。关键数据（FY2024）：

  • 收入 $19 亿（同比增长 737%，增速爆炸）
  • 约 25 万张 GPU（H100 为主，B200 快速补充）
  • 客户集中度极高：微软贡献 60%+ 收入，OpenAI、Meta 是前三大客户
  • 商业模式：长期租约（2-5 年），毛利率约 65%
  • 资本开支凶猛：2024 年 capex 约 $170 亿，几乎全部用来买 NVIDIA 卡

简单说，CoreWeave 是一个"GPU 二道贩子"——通过更激进的融资和快速部署能力，提供比 AWS/Azure 更便宜的 GPU 时租。它的生意建立在两个假设上：

  1. NVIDIA 愿意给它分配足够多的卡（供应端）
  2. 大客户愿意租而不是买/自建（需求端）

免费 token + 五层蛋糕战略，同时动摇了这两个假设。

冲击逻辑——三条战线

战线 1：需求端侵蚀——推理业务被平台化

CoreWeave 的客户用 GPU 主要做两件事：训练和推理。build.nvidia.com + NIM 直接把"推理"这个需求从裸 GPU 租赁层，抽象到了 API 调用层。

对中小客户：原来要租 CoreWeave 的 8 张 H100 做推理服务，现在直接调 NVIDIA API，省钱省事。

对中型客户：原来租 CoreWeave 自搭 vLLM，现在用 NIM 容器+NVIDIA 托管，迁移成本极低。

推理业务占 GPU 需求约 40-50%（2026 年还在上升）。这部分需求正在从"裸 GPU 租赁"迁移到"API/NIM 服务"——CoreWeave 作为中间商，变得多余。

战线 2：供给端施压——NVIDIA 掌握生杀大权

CoreWeave 的核心供应来自 NVIDIA。三年前，NVIDIA 为了让 Hopper 快速起量，给 CoreWeave 优先供货。现在形势变了——NVIDIA 自己做 DGX Cloud，CoreWeave 变成了 NVIDIA 自家服务业务的直接竞争对手。

NVIDIA 手里有三把刀可以施压：

  • 配额：把最新款 B200/Rubin 优先给 DGX Cloud，CoreWeave 拿到的是次新款
  • 价格：对 CoreWeave 收更高的单卡价格，压缩其毛利
  • 时间差：让 DGX Cloud 比 CoreWeave 早 3-6 个月拿到新品

这不是假设——2025 年底已有迹象显示 B200 的早期配额 DGX Cloud 拿走了更大比例。

战线 3：估值压缩——从"AI 云独角兽"到"GPU 二房东"

CoreWeave 2024 年上市时被当作"AI 云平台"讲故事，对标 AWS，给了 $350 亿估值、PE 倍数拉到 60x+。但如果 NVIDIA 做成了五层蛋糕，"AI 云平台"这个叙事的主角是 NVIDIA 自己，CoreWeave 只是"给 NVIDIA 算力饥饿者提供溢出容量的外包商"——这两种定位，估值倍数可能差 3-5 倍。

对 CoreWeave 投资者的含义

CoreWeave 的多头逻辑依赖"AI 算力继续爆炸式增长"——这一点短期没问题。但它的估值倍数依赖"CoreWeave 作为独立 AI 云平台"的叙事——这一点正在被 NVIDIA 自己拆台。

需要密切监控的信号：

  • DGX Cloud 的收入披露（NVIDIA 季度财报）——拐点是 run-rate 破 $50 亿/年
  • CoreWeave 的客户集中度变化——微软份额继续上升还是开始下降
  • NVIDIA 对 CoreWeave 的供货份额——是不是在被稀释
  • 新品（Rubin）首发分配——DGX Cloud vs CoreWeave 时间差
CoreWeave 的困境本质：它是 NVIDIA 崛起的直接受益者（从 NVIDIA 买卡转租获得超额回报），也是 NVIDIA 五层蛋糕完成后的直接受害者（自己的生意被 NVIDIA 内化）。它的命运由一个不受控的变量决定：NVIDIA 对它是合作还是竞争。

结语

免费送 token 这个动作很小，但连锁反应很大。

短期看，NVIDIA 用 1.5% 的净利润获得了：token 付费收入、硬件销售漏斗、开发者心智、生态锁定。账面上几年内就回本。

中期看，五层蛋糕中的"软件 + 服务"层开始真正跑起来——这是从"卖铲子"到"卖水电"的身份切换。

长期看，这场战略的真正赌注是估值模型重构——把 NVIDIA 从"周期性半导体公司"重新定位为"AI 平台公司"。参照苹果 2015-2024 年的剧本，这个重构可能带来 $5,000 亿级别的估值溢价。但 NVIDIA 当前 PE 已经 price-in 了一部分，所以关键不是"会不会成功"，而是"兑现的速度够不够快"。

副作用是：CoreWeave 这类中间商的生意模式被结构性动摇；云厂与 NVIDIA 的合作-竞争关系会更紧张；反垄断监管压力加大。

一句话：NVIDIA 免费发 token，是黄仁勋用硬件利润的零头，买未来 10 年的"算力收租权"——现在看起来像慈善，十年后看是史上最赚的生意。
